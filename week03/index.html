    DOM

    • document.body returns the body element of a web page
    • document.images returns a node list of all the images contained in the document.
    • document.links returns a node list of all the elements and elements that have an href attribute.
    • document.anchors returns a node list of all the elements that have a name attribute.
    • document.forms returns a node list of all the forms in the document.

    Other DOM Selectors to Remember

    • document.getElementById();
    • document.getElementsByTagName(); -- returns a list
    • document.getElementsByClassName(); --returns a list
    • document.querySelector(); -- searches by what the CSS selector would be
    • document.querySelectorAll();
    • document.createElement();
      • document.createTextNode();
      • .appendChild();
      • .removeChild();
      • .replaceChild();
    • .innerHTML; -- returns as raw html string

    Events

    • Attach an event to a node (ie, document.body)
    • document.body.addEventListener("click", doSomething)
    • Keypress event fires when a character is produced on screen, so modifier keys only fire keydown and keyup.
